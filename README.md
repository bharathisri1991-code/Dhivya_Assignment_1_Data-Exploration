# Dhivya_Assignment_1_Data-Exploration



Dhivya_Assignment_1_Data Exploration

This is a Data Analytics (Module 1) assignment focused on building foundational skills in Microsoft Excel. You are acting as a Data Analyst tasked with exploring a "Product Dataset" containing attributes like Product ID, Name, Brand, Quantity, Category, and Price.

Here are the formulas used in this assignment

Assignment 1 Data Exploration

1. Sum, Count, and Average 
SUM FUNCTION =SUM(D2:D35)
COUNT FUNCTION =COUNTA(B2:B35)
AVG FUNCTION =AVERAGE(D2:D35)
 

2. Minimum and Maximum Values
MIN FUNCTION  =MIN(D2:D35)
MAX FUNCTION  =MAX(D2:D35)
 
3. Logical Function – IF
IF FUNCTION =IF(D2>=500,"High price","Standard price")

4. Conditional Functions – SUMIF and COUNTIF
SUM IF FUNCTION =SUMIF(G2:G35, "Electronics", D2:D35)
COUNT IF FUNCTION  =COUNTIF(D2:D35,"<100")

5. Text Formatting – LEFT, RIGHT, MID
LEFT FORMATTING =LEFT(A2:A35,2)
RIGHT FORMATTING =RIGHT(A2:A35,2)
MID FORMATTING =MID(A2:A35,4,3)
